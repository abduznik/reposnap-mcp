Set up reposnap-mcp

Choose your AI assistant or MCP client below:

Claude Desktop App

Add this to your claude_desktop_config.json:

{ "mcpServers": { "reposnap": { "type": "url", "url": "https://reposnap-mcp.bacholate.workers.dev/mcp" } } }

Then restart Claude Desktop. You'll see "reposnap" in your tool list.

Cursor

Add this to your MCP configuration (usually in .cursor/mcp.json):

{ "mcpServers": { "reposnap": { "type": "url", "url": "https://reposnap-mcp.bacholate.workers.dev/mcp" } } }

Generic MCP Client / Windsurf

Direct endpoint URL:

https://reposnap-mcp.bacholate.workers.dev/mcp

Consult your MCP client's documentation for configuration details.

Pro tip: Once configured, just ask your AI assistant to "ingest the repository at [GitHub URL]" and you're done!

How it works

Point your MCP client to reposnap-mcp's free endpoint. Configuration takes under a minute — just paste the URL into your AI assistant's settings.

Call the ingest_repo tool with any public GitHub repository URL. No authentication required, works with private repos you have access to.

Get the full codebase injected into your AI's context instantly. File tree, all source code, configuration files — everything you need to understand, refactor, or analyze a project.

No manual copy-pasting. No file size limits. Just instant, complete repo context.

Tool reference

After configuring reposnap-mcp, you'll have access to the ingest_repo tool in your AI assistant.

Parameter Type Description
repo_url string A public GitHub repository URL (e.g., https://github.com/user/repo)

Returns: Complete file tree and full source code as plain text, ready for AI analysis.

Questions?

Is it free? Yes, completely. reposnap-mcp is open source (MIT) and the hosted endpoint is free to use.

Do I need an account? No. No authentication, no rate limiting for reasonable use.

Does it work with private repos? Yes, as long as you have access to the repository on GitHub.

Can I self-host? Absolutely. The source code is on GitHub — deploy it yourself using Cloudflare Workers, Docker, or any serverless platform.

What about my data? reposnap-mcp fetches public code from GitHub on demand. We don't store your data.
