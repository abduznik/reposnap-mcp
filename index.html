reposnap-mcp

Ingest any GitHub repo into your AI context. One MCP tool, zero setup.

View on GitHub ❤️ Sponsor

How it works

1

Point your MCP client

Configure Claude, Cursor, Windsurf, or any MCP client to use our endpoint.

2

Call ingest_repo

Pass any public GitHub URL and let the tool do the work.

3

Get full context

Receive the complete file tree and source code in your AI's context.

Install

Add this to your claude_desktop_config.json or your MCP client's configuration:

{ "mcpServers": { "reposnap": { "type": "url", "url": "https://reposnap-mcp.bacholate.workers.dev/mcp" } } }

Tool reference

Tool Input Output
ingest_repo repo_url (GitHub URL) File tree + full source as plain text
